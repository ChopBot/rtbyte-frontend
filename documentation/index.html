General usage

RTByte commands are always prefaced by a dash (-) on a server with default configuration. If this does not work, please contact the server staff of the Discord server you're trying to execute it on, as they may have changed it manually.

Parts of commands with [ ] around them are optional, whereas parts with < > are required.

User commands

COMMAND DESCRIPTION ALIASES
-userconf <set|show|remove|reset> <key> <value> Define per-user configuration.
-help [command] Displays a list of commands, or extended help for a specified command. commands
-info Provides bot information, such as a list of authors and libraries. details, what
-report <user> <reason> Reports mentioned user to the mod team, reason required. Screenshots appreciated. Append -s to report silently. r
-roles <list|join|leave|add|remove> [user] [role] A series of commands to allow you to join roles by yourself. roleme, team, squad
-quote <message ID> [channel] Quotes a message by ID. Be sure to mention the channel if the message you want to quote is not the same as the one you're sending the message in.
-joindate [user] Displays your account creation date along with the date you joined the current server you're on. Other users can be specified to fetch their account creation and server join dates. me, myself, age
-8ball <question>? Magic 8-ball, does exactly what the toy does, memes included. mirror, magic, conch
-choice <choices>, <choices> Makes a decision based off of the choices given, no matter how many you include. choose, decide
-coinflip Flips a coin. ðŸ™‚ for heads, ðŸ™ƒ for tails. coin, flip
-roll [amount of dice] [type of dice] Rolls a die. Optionally specify the amount of dice and their type. dice, d20

Moderation commands

COMMAND DESCRIPTION ALIASES
-userinfo [user] Displays user information. Returns your info if no other user is specified.
-serverinfo Displays server information. sinfo, guildinfo
-sendmsg <channel/user> <message> Sends a message to the specified channel or user using RTByte. sendmessage, message, msg
-purge [user] <amount> [all] [s] Removes X amount of messages, optionally sent by Y user. Append the word 'all' to ignore role hierarchy. p
-warn <user> <reason> Sends a formal warning to the mentioned user with the specified reason, and logs it. w
-mute <user> <reason> Mutes a mentioned user and logs the reason. m
-unmute <user> Unmutes mentioned user and logs it. um
-kick <user> <reason> Kicks a mentioned user and logs the reason. k
-softban <user> <reason> Kicks user from server and removes all their messages sent during the past 24 hours. sb
-ban <user> <reason> Bans a mentioned user and logs the reason. b
-vckick <user> <reason> Kicks a mentioned user from voice chat and logs the reason. vk, vck
-vcban <user> <reason> Bans a mentioned user from voice chat and logs the reason. vb, vcb
-vcunban <user> Unbans a mentioned user from voice chat. vub, vcub
