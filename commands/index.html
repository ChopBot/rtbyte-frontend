Enlarges a specified emoji. Standard emojis unfortunately do not work.

-info

Displays the link to invite RTByte to your server.

Displays your account creation date along with the date you joined the current server you're on. Other users can be specified to fetch their info.

-joindate [user]

lyrics

Fetches lyrics and information from Genius for you.

-lyrics <song>

ping

Runs a connection test to Discord.

-ping

prefix

Returns the prefix(es) for the current server.

-prefix

quote

Quotes a message by ID. Mention the channel if the message you want to quote is not the same as the one you're sending the message in.

-quote <message ID> [channel]

reminder

Sets a reminder for you, append -dm to have the reminder sent to you via DM.

-reminder <when> <message> [-dm]

roles

A series of commands to allow you to join roles by yourself.

-roles <list/join/leave/add/remove> [user] [role]

star

Fetch a random message from the starboard. Append top to get the top starred message instead.

-star [top]

weather

Fetch the current weather for a specified location.

-weather <location>

Command

Description

Usage

userinfo

Get information on a specified user. If no user is specified, it returns your info.

-userinfo [user]

channelinfo

Get information on a specified channel. If no channel is specified, it returns your current channel.

-channelinfo [channel]

roleinfo

Get information on a specified role. If no role is specified, it returns your highest role.

-roleinfo [role]

serverinfo

Displays server information.

-serverinfo

sendmsg

Sends a message to the specified channel or user using RTByte.

-sendmsg <channel or user> <message>

modhistory

Lists moderation actions taken against a user in the past. Refers to a specific case if a case ID is provided.

-modhistory [user] [case ID]

purge

Removes X amount of messages, optionally sent by Y user. Append -all to ignore the role hierarchy.

-purge [user] <amount> [-all] [-s]

warn

Warns a specified user and logs the reason.

-warn <user> [reason] [-s]

mute

Mutes a specified user and logs the reason.

-mute <user> [when] [reason] [-s]

unmute

Unmutes a specified user.

-unmute <user>

kick

Kicks a specified user and logs the reason.

-lyrics <song>

softban

Softbans a specified user and logs the reason.

-softban <user> [reason] [-s]

ban

Bans a specified user and logs the reason.

-ban <user> [when] [reason] [-s]

vckick

Kicks a specified user from voice chat and logs the reason.

-vckick <user> [reason] [-s]

vcban

Bans a specified user from voice chat and logs the reason.

-vcban <user> [when] [reason] [-s]

vcunban

Unbans a specified user from voice chat.

-vcunban <user>
